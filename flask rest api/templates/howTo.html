How to use this api

The ip can either be changed when asked in the prompt after starting the app.py
or it can be changed in the 'opcua_client.py' file.

The server can be launched in Terminal by running The command
'flask run'
from the flask rest api directory

By going to /api you will be able to see
the different api sub-sites and an explanation on how to use said sub-site
i.e get / post methodes
return types and expected types

When looking at /api you may be able to spot 'G' / 'S'
'G' means the sub-site only have getters were 'S' means the sub-site has Setters
